# Typing Master Pro

A modern, fast, and interactive typing master web app built with React, Vite, and Tailwind CSS.

## 🚀 Features

- **Real-time Typing Test:** Dynamic word generation and tracking.
- **WPM Calculation:** Accurate words per minute calculation.
- **Accuracy Tracking:** Monitor your typing precision.
- **Dark Mode:** Easy on the eyes for late-night typing.
- **Responsive Design:** Clean and centered UI across devices.
- **Smooth Animations:** Built with transitions for a polished feel.

## 🛠️ Tech Stack

- **React:** Frontend framework.
- **Vite:** Next-generation frontend tooling.
- **Tailwind CSS:** Utility-first CSS framework.
- **Lucide React:** Icon library.

## 📦 Installation & Setup

1. **Clone the project** (if applicable) or enter the project directory:
   ```bash
   cd typing-master-pro
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## ⌨️ How to Play

1. Head to the **Test** page.
2. Start typing the words highlighted in the box.
3. The timer starts automatically on your first keystroke.
4. Try to reach the highest WPM with maximum accuracy!

Enjoy improving your typing speed! 🚀
